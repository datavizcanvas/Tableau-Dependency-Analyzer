Tableau Dependency Analyzer

A Streamlit application that analyzes Tableau workbook (.twb/.twbx) files to:
- Extract all fields (default, calculated, parameters)
- Export formulas of calculated fields
- Map dependencies between fields
- Calculate impact metrics and risk levels
- Generate interactive HTML reports with dependency diagrams
- Export data to CSV/Excel

Features: Mermaid.js diagrams, click-to-highlight dependencies, Excel downloads, PNG export, zoom controls.
